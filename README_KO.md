**웹사이트**: [ethskills.com](https://ethskills.com/) | **GitHub**: [austintgriffith/ethskills](https://github.com/austintgriffith/ethskills) | **리서치**: [austintgriffith/ethskills-research](https://github.com/austintgriffith/ethskills-research)

각 스킬은 마크다운 파일입니다. AI 에이전트에 URL을 제공하면 즉시 이더리움 지식이 교정됩니다. 설치 불필요.

```bash
# Claude Code 플러그인으로 추가
/plugin marketplace add austintgriffith/ethskills
/plugin install ethskills

# 또는 아무 AI 에이전트에 URL을 직접 제공
```

## Uniswap AI — 온체인 에이전트를 위한 7가지 스킬

> Uniswap이 핵심 Uniswap 프로토콜 작업에 구조화된 접근을 제공하는 7가지 새로운 스킬(Skills)을 출시했습니다. 온체인 에이전트 워크플로우의 시작점입니다.

**GitHub**: [Uniswap/uniswap-ai](https://github.com/Uniswap/uniswap-ai) | **문서**: [ul-ai-docs.vercel.app](https://ul-ai-docs.vercel.app/)

### 빠른 시작

```bash
# Skills CLI (모든 에이전트)
npx skills add Uniswap/uniswap-ai

# Claude Code 마켓플레이스
/plugin marketplace add uniswap/uniswap-ai

# 개별 플러그인 설치
/plugin install uniswap-hooks      # v4 훅 개발
/plugin install uniswap-trading    # 스왑 통합
/plugin install uniswap-cca        # CCA 옥션
/plugin install uniswap-driver     # 스왑 & 유동성 플래닝
/plugin install uniswap-viem       # EVM 통합 (viem/wagmi)
```

### 7가지 스킬 개요

| # | 스킬 | 플러그인 | 설명 |
|:-:|:------|:-------|:------------|
| 1 | **Swap Planner** (스왑 플래너) | uniswap-driver | 토큰 스왑을 계획하고 Uniswap 인터페이스에서 실행할 딥 링크를 생성합니다. 모든 지원 체인에서 토큰 탐색, 가격 리서치, 리스크 평가를 지원합니다. |
| 2 | **Liquidity Planner** (유동성 플래너) | uniswap-driver | 가격 범위 제안, 풀 분석(APY, TVL, 거래량)과 함께 LP 포지션을 계획하고 v2/v3/v4 포지션 생성을 위한 딥 링크를 생성합니다. |
| 3 | **Swap Integration** (스왑 통합) | uniswap-trading | Trading API, Universal Router, 직접 컨트랙트 호출을 통해 Uniswap 스왑을 프론트엔드, 백엔드, 스마트 컨트랙트에 통합합니다. CLASSIC, DUTCH_V2, PRIORITY 라우팅을 지원합니다. |
| 4 | **V4 Security Foundations** (V4 보안 기초) | uniswap-hooks | Uniswap v4 훅 구축을 위한 보안 우선 가이드. NoOp 러그풀 공격, 14개 권한 플래그 리스크 매트릭스, 델타 회계, 배포 전 감사 체크리스트를 다룹니다. |
| 5 | **CCA Configurator** (CCA 구성기) | uniswap-cca | 연속 청산 옥션(CCA) 파라미터의 대화형 구성 — 가격 책정(Q96 포맷), 공급 스케줄, 블록 타이밍, 멀티체인 배포 설정. |
| 6 | **CCA Deployer** (CCA 배포기) | uniswap-cca | Factory 패턴과 CREATE2를 사용하여 CCA 스마트 컨트랙트를 배포합니다. 일관된 주소를 보장합니다. 프라이빗 키 보안 가이드 및 배포 후 검증을 포함합니다. |
| 7 | **viem Integration** (viem 통합) | uniswap-viem | viem과 wagmi를 사용한 EVM 블록체인 통합 — 클라이언트 설정, 컨트랙트 읽기/쓰기, 트랜잭션 서명, 이벤트 모니터링, React 훅. |

### 아키텍처

```
uniswap-ai/ (Nx 모노레포)
├── packages/plugins/
│   ├── uniswap-hooks/     → V4 보안 기초
│   ├── uniswap-trading/   → 스왑 통합
│   ├── uniswap-cca/       → CCA 구성기 + 배포기
│   ├── uniswap-driver/    → 스왑 플래너 + 유동성 플래너
│   └── uniswap-viem/      → viem 통합
├── evals/                 → Promptfoo 평가 스위트 (>=85% 통과율)
└── docs/                  → VitePress 문서
```

### 핵심 설계 원칙

- **에이전트 비종속적(Agent-Agnostic)**: 모든 스킬이 Claude Code뿐 아니라 모든 LLM 코딩 에이전트에서 작동
- **평가 주도(Eval-Driven)**: 모든 스킬에 자동화된 평가(Promptfoo) 적용 — 평가는 AI 도구에게 테스트가 코드에게 하는 것과 같은 역할
- **딥 링크 패턴**: Swap Planner와 Liquidity Planner는 파라미터가 사전 입력된 Uniswap URL을 생성
- **보안 우선**: V4 훅 스킬은 완전한 위협 모델, NoOp 공격 탐지, 리스크 스코어링(0-33)을 포함
