## Ethereum Skills for AI Agents (ETHSKILLS)

> The missing knowledge between AI agents and production Ethereum. LLMs think gas is 30 gwei (reality: 0.05), hallucinate contract addresses, and don't know about ERC-8004 or EIP-7702. ETHSKILLS fixes that.

**Website**: [ethskills.com](https://ethskills.com/) | **GitHub**: [austintgriffith/ethskills](https://github.com/austintgriffith/ethskills) | **Research**: [austintgriffith/ethskills-research](https://github.com/austintgriffith/ethskills-research)

### How It Works

Each skill is a markdown file. Give any URL to your AI agent — it reads it and instantly corrects its Ethereum knowledge. No installation required.

```bash
# Add to Claude Code as a plugin
/plugin marketplace add austintgriffith/ethskills
/plugin install ethskills

# Or give the URL directly to any AI agent
https://ethskills.com/SKILL.md
```

### Available Skills (18 Skills)

## Uniswap AI — 7 Skills for Onchain Agents

> Uniswap launched 7 new Skills providing structured approaches to core Uniswap protocol operations. A starting point for onchain agent workflows.

**GitHub**: [Uniswap/uniswap-ai](https://github.com/Uniswap/uniswap-ai) | **Docs**: [ul-ai-docs.vercel.app](https://ul-ai-docs.vercel.app/)

### Quick Start

```bash
# Skills CLI (any agent)
npx skills add Uniswap/uniswap-ai

# Claude Code Marketplace
/plugin marketplace add uniswap/uniswap-ai

# Install individual plugins
/plugin install uniswap-hooks      # v4 hook development
/plugin install uniswap-trading    # Swap integration
/plugin install uniswap-cca        # CCA auctions
/plugin install uniswap-driver     # Swap & liquidity planning
/plugin install uniswap-viem       # EVM integration (viem/wagmi)
```

### 7 Skills Overview

| # | Skill | Plugin | Description |
|:-:|:------|:-------|:------------|
| 1 | **Swap Planner** | uniswap-driver | Plan token swaps and generate deep links to execute in the Uniswap interface. Supports token discovery, price research, and risk assessment across all supported chains. |
| 2 | **Liquidity Planner** | uniswap-driver | Plan LP positions with price range suggestions, pool analytics (APY, TVL, volume), and generate deep links for v2/v3/v4 position creation. |
| 3 | **Swap Integration** | uniswap-trading | Integrate Uniswap swaps into frontends, backends, and smart contracts via Trading API, Universal Router, or direct contract calls. Supports CLASSIC, DUTCH_V2, and PRIORITY routing. |
| 4 | **V4 Security Foundations** | uniswap-hooks | Security-first guide for building Uniswap v4 hooks. Covers the NoOp rug pull attack, 14 permission flags with risk matrix, delta accounting, and pre-deployment audit checklist. |
| 5 | **CCA Configurator** | uniswap-cca | Interactive configuration of Continuous Clearing Auction (CCA) parameters — pricing (Q96 format), supply schedules, block timing, and multi-chain deployment settings. |
| 6 | **CCA Deployer** | uniswap-cca | Deploy CCA smart contracts via the Factory pattern with CREATE2 for consistent addresses. Includes private key security guide and post-deployment verification. |
| 7 | **viem Integration** | uniswap-viem | EVM blockchain integration using viem and wagmi — client setup, contract reads/writes, transaction signing, event monitoring, and React hooks. |

### Architecture

```
uniswap-ai/ (Nx monorepo)
├── packages/plugins/
│   ├── uniswap-hooks/     → V4 Security Foundations
│   ├── uniswap-trading/   → Swap Integration
│   ├── uniswap-cca/       → CCA Configurator + Deployer
│   ├── uniswap-driver/    → Swap Planner + Liquidity Planner
│   └── uniswap-viem/      → viem Integration
├── evals/                 → Promptfoo evaluation suites (>=85% pass rate)
└── docs/                  → VitePress documentation
```

### Key Design Decisions

- **Agent-Agnostic**: All skills work with any LLM coding agent, not just Claude Code
- **Eval-Driven**: Every skill has automated evaluations (Promptfoo) — evals are to AI tools what tests are to code
- **Deep Link Pattern**: Swap Planner and Liquidity Planner generate URLs that open Uniswap with pre-filled parameters
- **Security-First**: V4 hooks skill includes full threat model, NoOp attack detection, and risk scoring (0-33)
